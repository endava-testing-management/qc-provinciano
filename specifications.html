User Story: Search Provinces

Description

  • As a user I'll be able to search information aboout the Argentinian Provinces

AC: acceptance criteria

  • The title page should be "Province Searcher"
  • The sub title page should be "Search data about Provinces of Argentina"
  • As a user Im able to search by diferent filters
    • By region, should be a dropdown with the all and none options included. The options should be ordered
    • By province name, should a text empty be default
  • As a user I search and obtain the following result
    • A list of results is displayed corresponding with the used filters
    • Columns
      • Region corresponding to the province
      • Name of the province
      • Province Capital city
      • Province Population
      • Province extention in km
    • The restults are alphabetically ordered by default
    • The results contains:
      • The provinces corresponding to the selected province
      • The provinces wich Name match with text entered in Name
  • As a user Im able to order by any of the columns
  • The information should match with the one in “Provinces of Argentina” from wikipedia (https://en.wikipedia.org/wiki/Provinces_of_Argentina)
  • If there are not matching results the following message “There are none province with the selected filters”
