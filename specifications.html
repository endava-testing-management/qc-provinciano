User Story

Search Provinces

Description

  • As an Administrator User I want to be able to search information about the Argentinian Provinces.

AC: acceptance criteria

  • The title of the page is "Province Searcher", capitalized and in orange color."
  • The page subtitle is "Search data about Provinces of Argentina", black color, italic.
  • As an Administrator User I'm able to search by different filters:
    • Region: Dropdown with a list of regions (Centro, Norte Grande Argentino, Nuevo Cuyo and Patagónica). Also options 'All' and 'None' are available. The options are ordered alphabetically ascendant. ‘All’ option by default.
    • Name: text box. Default option is empty
  • Search button: ‘search!’, orange background, white characters.
  • As an Administrator User I want to make a search and obtain the results that matches with criteria entered.
  • Search Results are displayed on a table with the following columns:
    • Region: Name of the region
    • Name: Name of the province
    • Capital: Capital city of the Province
    • Population: Province Population
    • KM2: Province extension in km2.
  • The search results are ordered alphabetically by province name by default.
  • The search results contains:
    • The provinces corresponding to the selected region.
    • The provinces where Name match with criteria entered in Name field.
    • The provinces that matches with Region and Name entered.
  • As an Administrator User I'm able to sort by any of the columns (ascendant and descendant).
  • The information should match with the one in “Provinces of Argentina” from wikipedia (https://es.wikipedia.org/wiki/Regiones_integradas_(Argentina))
  • If there are not matching results the following message “There are none province with the selected filters”
